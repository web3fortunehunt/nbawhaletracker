# PolySportsLab 巨鲸追踪器 - 产品文档

## 1. 产品概述

**PolySportsLab 巨鲸追踪器 (Whale Tracker)** 是一款专为体育预测市场交易者设计的专业分析工具。它实时追踪 PolySportsLab/Polymarket 上的“聪明钱”（巨鲸/盈利交易者）动向，帮助用户识别高胜率的交易机会和市场趋势。

为了确保高可用性、低延迟和全球访问，本系统目前正迁移至基于 Cloudflare 的云原生架构。

## 2. 核心功能

### 2.1 实时数据看板 (Dashboard)
*   **市场总览**：一目了然地查看总交易量、流动性和平均巨鲸集中度等关键指标。
*   **趋势可视化**：提供交互式图表，展示以下历史数据：
    *   **交易量与流动性**：追踪资金流入市场的动态。
    *   **巨鲸集中度**：观察大户是否正在吸筹或建仓。
    *   **价格历史**：将价格波动与巨鲸活动进行关联分析。
*   **巨鲸方向指标**：清晰指示“聪明钱”当前的下注方向（YES、NO 或中立）。

### 2.2 数据智能
*   **巨鲸集中度 (Whale Concentration)**：独家指标，计算头部盈利钱包持有的市场份额百分比。
*   **聪明钱方向 (Smart Money Direction)**：分析头部持有者是净做多 (Yes) 还是净做空 (No)。
*   **Top 10 持仓分析**：详细拆解每个市场中最大持仓者的仓位情况。

### 2.3 自动化数据采集
*   **定时更新**：系统每隔 X 分钟（可配置）自动从 PolySportsLab 获取最新数据。
*   **历史归档**：所有数据永久存储，支持回测和长期趋势分析。

## 3. 用户工作流

1.  **访问**：用户访问部署好的 Web 应用（例如 `https://tracker.your-domain.com`）。
2.  **筛选**：用户选择特定的市场（如“湖人 vs. 勇士”）或日期范围进行分析。
3.  **分析**：
    *   查看 **总览卡片** 获取即时信号。
    *   使用 **趋势图表** 识别吸筹模式。
    *   深入 **原始数据表** 查看精确数值。
4.  **决策**：利用这些洞察在 Polymarket 上进行知情交易。

## 4. 部署与可用性

*   **平台**：托管于 Cloudflare 全球边缘网络。
*   **可用性**：Serverless 架构保障 24/7 全天候在线。
*   **性能**：边缘缓存确保无论用户身在何处都能秒级加载。
